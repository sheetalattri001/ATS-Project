AI-Powered Recruitment & ATS Platform

An intelligent Applicant Tracking System (ATS) designed to streamline candidate screening and ranking using AI and NLP technologies.

 Tech Stack
* Frontend: React.js
* Backend: FastAPI (Python)
* Database: PostgreSQL

Features
* Resume Parsing & Ranking: Automatically extracts and ranks candidate information from resumes.
* Skill Matching: Efficiently matches candidate skills with job descriptions using NLP.
* Interview Scheduling: Simplifies the management and scheduling of candidate interviews.

 How to Run
1. Clone the repository:
   git clone https://github.com/sheetalattri001/ATS-Project.git
2. Install the required dependencies:
   pip install -r requirements.txt
3. Start the backend server:
   uvicorn main:app --reload
4. Start the frontend application:
   npm start
